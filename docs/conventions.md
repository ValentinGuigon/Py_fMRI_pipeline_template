# Naming Conventions

All patterns below are enforced by `scripts/run/run_pipeline.sh`. These are the authoritative rules for locating or creating any file in the pipeline.

---

## Input files

| File | Pattern |
|------|---------|
| Model JSON | `fitlins_models/{task_group}/{stem}_smdl.json` |
| Config | `fitlins_configs/{task_group}/{stem}.cfg` |

---

## Output suffix

`{stem}_s{kernel_mm}` — e.g. `tmth_visual_vs_baseline_s4`

Derived at runtime from `MODEL` and the kernel in `SMOOTH` (first colon-separated field of `SMOOTH=4:run:iso`).

---

## FitLins derivatives

| Path | Pattern |
|------|---------|
| Output directory | `fitlins_derivatives/{task_group}/{output_suffix}/` |
| Run-level stat maps | `fitlins_derivatives/{task_group}/{output_suffix}/node-{runNode}/` |
| Group-level stat maps | `fitlins_derivatives/{task_group}/{output_suffix}/node-{groupNode}/` |
| HTML report | `fitlins_derivatives/{task_group}/{output_suffix}/reports/model-{Name}.html` |

`{Name}` = the `Name` field in the model JSON (CamelCase, no spaces, no hyphens).
`{runNode}` / `{groupNode}` = bare node names from the model JSON (e.g. `runLevel`, `subjectLevel`, `datasetLevel`).

---

## Threshold suffix

Built from config values by `build_thr_suffix()` in `run_pipeline.sh`:

```
{thr_mode}[_p{P_UNC} | _t{THR_FIXED} | _a{ALPHA}][_2s | _1s][_df{n}][_svc]
```

| Config | Appended segment |
|--------|-----------------|
| `THR_MODE=p-unc` | `_p{P_UNC}` |
| `THR_MODE=fixed` | `_t{THR_FIXED}` |
| `THR_MODE=fdr\|bonferroni\|ari` | `_a{ALPHA}` |
| `TWO_SIDED=1` | `_2s` |
| `TWO_SIDED=0` | `_1s` |
| `DF_OVERRIDE` set | `_df{n}` |
| `ROI_MASK` set | `_svc` |

Examples: `p-unc_p0.01_2s`, `fixed_t4.0_1s`, `fdr_a0.05_2s_svc`

---

## Figures

| Path | Pattern |
|------|---------|
| Figure directory | `figures/{task_group}/{output_suffix}/{node_name}_{thr_suffix}/` |
| Glass brain | `{figure_dir}/{tag}__glass.png` |
| Slice display | `{figure_dir}/{tag}__slices.png` |
| Interactive 3D | `{figure_dir}/{tag}__3d.html` (only with `--view3d`) |
| Cluster table | `{figure_dir}/{tag}__clusters.tsv` (only with `CLUSTER_TABLE=1`) |
| Manifest | `{figure_dir}/manifest.tsv` |

`manifest.tsv` is the authoritative record of every stat map processed: threshold applied, threshold value, and paths to all output figures.

Example figure directory: `figures/tmth/tmth_visual_vs_baseline_s4/datasetLevel_p-unc_p0.01_2s/`

---

## Reports

`reports/{task_group}/{output_suffix}__{thr_suffix}.pdf`

Double underscore separates output suffix from threshold suffix.

Example: `reports/tmth/tmth_visual_vs_baseline_s4__p-unc_p0.01_2s.pdf`

---

## Analysis indexes

`docs/indexes/{task_group}_analysis_index.[json|md]`

Generated by `scripts/report/build_analysis_index.py`. One pair of files per task group.

---

## SLURM logs

`slurm_logs/{task_group}/fitlins_{stem}_{YYYYMMDD_HHMMSS}[.sbatch | _{jobid}.out | _{jobid}.err]`
